Command Description Usage
Help Displays the help command. !help
Play This command will play a song. If a song is currently playing it will add it to the queue. You can type a song name or paste a link to the YouTube video. !play [song name or url]
Skip Will skip the current song. !skip
Stop Will stop all music and delete the queue. !stop
Now Playng Displays what song is currently playing. !np
Lyrics Will get the currently playing songs lyrics. Lyrics are provided by Genius. !lyrics
Lyrics Will get the lyrics for the provided song. Lyrics are provided by Genius. !lyrics [song name]
Queue Displays current queue. !queue
Volume Set's the volume. Use a number between 1 and 5. !volume
Invite Sends an invite link for the bot. !invite
